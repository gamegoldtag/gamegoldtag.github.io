새로운 플랫폼과 언어를 꾸준히 익히고, 실제 업무에 적용하는 개발자입니다. 이 사이트는 특정 회사나 제품 홍보가 아닌, 실무에서 검증한 기술 선택과 구현 과정을 기록하는 기술 블로그입니다.

경험 영역

Windows, 웹, 데스크톱, 클라우드 환경을 모두 다뤄왔고, C++, C#, Rust, TypeScript, React, Vue, PHP 등 여러 언어와 프레임워크를 프로젝트 상황에 맞춰 사용해 왔습니다.

운영 원칙

이 사이트의 글은 직접 작성한 실무 경험 기반 콘텐츠만 게시합니다. 단순 개념 요약보다 "왜 이 선택을 했는가", "무엇이 실패했고 어떻게 고쳤는가"를 중심으로 작성합니다.

다루는 주제

Electron, Rust, TypeScript, React, 프론트엔드 성능 최적화, 개발 생산성 자동화, 아키텍처 설계, 디버깅 방법론을 중점적으로 다룹니다.
