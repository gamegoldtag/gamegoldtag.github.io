AI가 만들어주는 나만의 무료 웹사이트

게시일: 2025년 11월 22일 · 11분 읽기

"내 웹사이트 하나 갖고 싶은데 개발은 모르고, 돈도 부담된다"는 고민이 있습니다. 2025년 기준으로는 AI 도구를 활용해 초기 비용 없이 개인 사이트를 시작할 수 있습니다. 이 글에서는 AI로 웹사이트를 만들고 무료 호스팅에 올리는 전체 흐름을 정리합니다.

AI가 코드를 짜준다는 말, 어디까지 맞을까?

ChatGPT, Claude 같은 도구에 원하는 화면과 기능을 구체적으로 설명하면 HTML/CSS/JS 초안을 빠르게 만들 수 있습니다. 다만 결과물은 "완성본"이 아니라 "초안"으로 보는 게 정확합니다. 실제 공개 전에는 내용 검수, 오탈자 점검, 모바일 확인이 꼭 필요합니다.

전체 흐름 한눈에 보기

  1. AI에게 원하는 사이트 설명
  2. 코드 생성 결과 확인
  3. index.html로 저장
  4. 무료 호스팅에 업로드
  5. 도메인/메타데이터 정리 후 공개

1단계: AI에게 구체적으로 요청하기

좋은 프롬프트일수록 수정 횟수가 줄어듭니다. 테마, 메뉴 구성, 모바일 대응, 파일 구조를 함께 지정하세요.

  • 좋은 예: "포트폴리오 사이트, 다크 테마, 홈/소개/작업물/연락처 메뉴, 모바일 반응형, HTML 한 파일"
  • 아쉬운 예: "웹사이트 만들어줘"

2단계: 코드 저장 후 반복 수정

생성된 코드를 복사해 index.html로 저장하고 브라우저에서 확인합니다. 이후 "배경 더 어둡게", "폰트 변경", "카드 hover 추가"처럼 작은 단위로 다시 요청하면 품질이 빠르게 올라갑니다.

3단계: 무료 호스팅에 배포

서비스특징난이도
GitHub Pages무료, HTTPS 기본, 버전관리와 궁합 좋음중
Cloudflare Pages무료, 글로벌 CDN, 빠른 배포중
Netlify무료 플랜, UI 중심 배포가 쉬움하

추천 기준: 초보자면 Netlify, 코드 이력 관리까지 하려면 GitHub Pages가 편합니다.

GitHub Pages 배포 5단계

  1. GitHub에 새 저장소 생성
  2. index.html 업로드 및 커밋
  3. Settings → Pages에서 Branch 선택
  4. 배포 URL 발급 확인
  5. 모바일/데스크톱 최종 점검 후 공개

어떤 사이트에 적합한가?

  • 개인 포트폴리오, 소개 페이지, 이력서 페이지
  • 가게/업체 안내, 메뉴/가격표, 행사 안내 페이지
  • 프로젝트 데모, 기술 문서, 오픈소스 소개 페이지

한계와 주의사항

  • 로그인/결제 같은 복잡한 기능은 별도 백엔드가 필요합니다.
  • AI 생성 코드에는 버그가 있을 수 있으니 테스트가 필수입니다.
  • 이미지/문구의 저작권과 개인정보 포함 여부를 반드시 확인하세요.
요금제, 무료 한도, 서비스 정책은 시점에 따라 변경될 수 있으므로 배포 전 공식 문서를 확인하는 것이 안전합니다.

마무리

핵심은 "완벽한 코드를 한 번에 받기"가 아니라 "AI 초안을 빠르게 만들고 사람이 방향을 잡아 완성하는 것"입니다. 이 흐름만 익히면 비개발자도 수월하게 개인 웹사이트를 운영할 수 있습니다.
