돈 없이 내 웹사이트 만드는 법 - AI가 다 해줌

게시일: 2025년 11월 1일 · 10분 읽기

웹사이트 제작은 예전보다 훨씬 저렴해졌고, 초기 단계는 사실상 무료 조합으로 시작할 수 있습니다. 핵심은 "AI로 제작 초안" + "무료 디자인/이미지" + "무료 호스팅" 조합을 현실적으로 쓰는 것입니다.

비용이 발생하는 구간부터 나누기

항목유료 방식무료 대체
웹사이트 제작외주/개발자 의뢰AI 초안 생성 (Claude/ChatGPT)
디자인 작업유료 디자인 툴무료 플랜(Figma/Canva)
이미지 소스유료 스톡Unsplash / Pexels
호스팅월 과금형 서버Netlify / GitHub Pages / Cloudflare Pages
도메인연 과금초기엔 무료 서브도메인 사용

무료 도구 조합 가이드

제작: Claude, ChatGPT 모두 초안 제작에 충분합니다. 문서 구조와 UI 요구사항을 구체적으로 전달하면 결과가 안정적입니다.

이미지: Unsplash/Pexels에서 라이선스를 확인한 이미지를 사용하고, 필요하면 Canva 무료 플랜으로 크기와 비율을 맞춥니다.

호스팅: 입문자는 Netlify, 버전 관리 중심이면 GitHub Pages, 글로벌 성능 중심이면 Cloudflare Pages가 편합니다.

무료 서브도메인으로 먼저 시작하기

yoursite.netlify.app
yourname.github.io
yoursite.pages.dev

초기에는 위 주소로도 충분히 공개 테스트가 가능합니다. 사이트 방향이 확정되면 커스텀 도메인을 연결해 브랜드 신뢰도를 높이면 됩니다.

추천 조합

  • 가장 쉬운 시작: Claude + Unsplash + Netlify
  • 개발자 친화적: Claude + Canva + GitHub Pages
  • 속도/엣지 배포 중시: ChatGPT + Pexels + Cloudflare Pages

비용 비교는 이렇게 보는 게 정확함

외주 제작은 초기 제작비와 유지 비용이 함께 발생할 수 있습니다. 무료 조합은 초기 비용을 크게 줄여주지만, 기능 복잡도가 올라가면 결국 유료 인프라나 백엔드가 필요합니다.

"완전 0원"은 정적 소개 사이트 기준에 가깝습니다. 로그인/결제/대규모 트래픽/DB가 필요한 서비스는 별도 비용을 고려해야 합니다.

무료 조합의 한계

  • 회원가입/로그인/결제 기능은 추가 백엔드가 필요
  • 대규모 트래픽에서는 무료 한도 확인 필요
  • AI 생성 코드의 품질 검수(접근성, 모바일, 보안 헤더) 필요

마무리

핵심은 "무조건 무료"가 아니라 "초기 비용 없이 빠르게 검증"입니다. 아이디어 단계에서는 무료 조합으로 충분히 시작할 수 있고, 성과가 나오면 필요한 부분만 유료로 확장하는 전략이 가장 현실적입니다.
