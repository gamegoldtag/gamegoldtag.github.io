Cloudflare Pages - GitHub Pages보다 빠른 무료 호스팅?

게시일: 2025년 10월 10일 · 11분 읽기

Cloudflare Pages는 전 세계 엣지 네트워크를 기반으로 정적 사이트를 배포할 수 있는 서비스입니다. 무료 플랜만으로도 시작이 가능하고, Direct Upload와 Git 연동을 모두 지원합니다.

Cloudflare Pages가 주목받는 이유

핵심은 엣지 배포입니다. 방문자와 가까운 위치에서 콘텐츠를 제공하기 때문에 지역에 따라 체감 성능이 좋아질 수 있습니다. 특히 전 세계 방문자를 대상으로 하는 사이트에서 이점이 큽니다.

트래픽/빌드 횟수/기능 한도는 시점별로 달라질 수 있으므로, 적용 전 Cloudflare 공식 플랜 안내를 확인하세요.

준비물

  • Cloudflare 계정
  • 배포할 사이트 폴더(index.html 포함)
  • 선택 사항: GitHub 저장소(자동 배포용)

배포 방법 2가지

  1. Direct Upload: 폴더를 직접 올려 즉시 배포
  2. Git 연동: push 시 자동 빌드/배포

처음에는 Direct Upload가 빠르고, 운영 단계에서는 Git 연동이 편합니다.

Direct Upload (초기 배포)

  1. Cloudflare 로그인 후 Workers & Pages 이동
  2. Create 클릭 후 Pages 생성 시작
  3. Direct Upload 선택, 프로젝트 이름 지정
  4. 사이트 폴더 업로드 후 Deploy 실행
예: project-name.pages.dev

GitHub 연동 (자동 배포)

  1. Connect to Git 선택 후 저장소 연결
  2. 정적 HTML은 빌드 설정 없이 진행 가능
  3. React/Vue는 빌드 명령과 output 디렉터리 지정
  4. Save and Deploy 후 push마다 자동 반영

마무리

Cloudflare Pages는 무료 시작, 빠른 배포, Git 자동화까지 균형이 좋은 선택지입니다. 특히 방문자 지역이 다양한 사이트라면 후보군에 반드시 넣을 만합니다.
