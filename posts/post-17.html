개발자 없이 AI로 홈페이지 만들고 무료로 올리기

게시일: 2025년 10월 25일 · 12분 읽기

이 글은 "AI로 홈페이지 만들기"를 처음 시도하는 분을 위한 전체 실행 가이드입니다. 목표는 소개형 단일 페이지를 만들고, 무료 호스팅으로 실제 공개 URL까지 얻는 것입니다.

오늘 목표

목표: 소개 홈페이지 1개
예상 시간: 약 30분
비용: 시작 단계 무료
결과: 외부에서 접속 가능한 URL 확보

준비물 체크

  • PC 또는 노트북
  • 이메일 계정(서비스 가입용)
  • 소개 문구(이름, 하는 일, 연락 채널)

1단계: Claude에서 초안 생성

claude.ai에서 새 대화를 열고 아래 구조로 요청합니다. 핵심은 섹션 순서와 디자인 조건을 함께 명시하는 것입니다.

이름: [홍길동]
소개: [프리랜서 번역가]
섹션: 자기소개 / 제공 서비스 3개 / 연락처
디자인: 깔끔한 톤, 흰 배경, 남색 포인트
요구: 모바일 대응, HTML/CSS/JS 한 파일

2단계: 코드 저장

생성된 코드를 복사해 index.html로 저장합니다.

  • Windows: 저장 시 파일 형식 "모든 파일" 선택
  • macOS: TextEdit를 일반 텍스트 모드로 전환 후 저장

3단계: 브라우저 확인 후 수정 반복

index.html을 열고 수정할 부분을 묶어서 요청하면 효율이 좋습니다.

1) 상단 제목 폰트 확대
2) 서비스 섹션 배경을 연한 회색으로 변경
3) 하단 연락 버튼 추가
4) 전체 여백 확대

수정 코드를 다시 저장하고 새로고침하며 반복합니다.

4단계: Netlify 가입

netlify.com에서 계정 생성 후 대시보드로 이동합니다.

5단계: 폴더 단위 업로드

index.html이 들어 있는 폴더를 드래그앤드롭합니다. 이미지/아이콘 파일이 있다면 같은 폴더에 함께 있어야 정상 표시됩니다.

배포가 완료되면 *.netlify.app 주소가 발급됩니다.

6단계: URL 공유 및 이름 정리

발급된 주소를 SNS, 명함, 이메일 서명에 사용합니다. 필요하면 Netlify의 사이트 이름 변경 기능으로 URL을 더 읽기 쉽게 조정할 수 있습니다.

자주 막히는 문제

  • 파일 더블클릭 시 메모장으로 열림: 기본 앱을 브라우저로 변경
  • 배포 후 화면 깨짐: 파일만이 아니라 폴더 전체 업로드
  • 수정 반영 안 됨: 수정 후 재업로드하거나 Deploy 내역에서 새 배포 실행

주의사항

  • AI가 만든 문구에 개인정보가 과도하게 들어가지 않도록 확인하세요.
  • 이미지와 아이콘의 라이선스를 확인하세요.
  • 무료 플랜 정책은 시점에 따라 변경될 수 있습니다.
핵심은 한 번에 완벽한 결과를 만들려는 것이 아니라, 30분 내 공개 가능한 버전을 만든 뒤 점진적으로 개선하는 방식입니다.

마무리

흐름은 단순합니다. AI로 초안 생성 → 로컬 확인 → 수정 반복 → 무료 배포. 이 과정만 익히면 개발 경험이 없어도 소개형 홈페이지를 스스로 운영할 수 있습니다.
