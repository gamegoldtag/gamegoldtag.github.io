개발자 없이 AI로 홈페이지 만들고 무료로 올리기

게시일: 2025년 10월 25일 · 14분 읽기

이 글은 "AI로 홈페이지 만들기"를 처음 시도하는 분을 위한 전체 실행 가이드입니다. 목표는 소개형 단일 페이지를 만들고, 무료 호스팅으로 실제 공개 URL까지 얻는 것입니다. 코딩 지식이 전혀 없어도 따라올 수 있도록 각 단계를 상세하게 설명합니다.

오늘 목표

목표: 소개 홈페이지 1개
예상 시간: 약 30분
비용: 시작 단계 무료
결과: 외부에서 접속 가능한 URL 확보

준비물 체크

  • PC 또는 노트북
  • 이메일 계정 (서비스 가입용)
  • 소개 문구 (이름, 하는 일, 연락 채널)
  • 사용할 이미지 (선택사항, 없으면 텍스트 중심으로 시작 가능)

1단계: Claude에서 초안 생성 (5분)

claude.ai 또는 ChatGPT를 열고 아래 형식으로 요청합니다. 핵심은 섹션 순서와 디자인 조건을 함께 명시하는 것입니다.

이름: [홍길동]
소개: [프리랜서 번역가 · 영어/일본어 전문]
섹션: 자기소개 / 제공 서비스 3개 / 대표 작업 사례 / 연락처
디자인: 깔끔한 톤, 흰 배경, 남색 포인트
요구:
  - 모바일 반응형
  - HTML/CSS/JS 한 파일
  - Pretendard 폰트 CDN
  - 한국어

요청 후 코드가 생성되면 전체 선택 후 복사합니다.

2단계: 코드 저장 (3분)

복사한 코드를 index.html 파일로 저장합니다.

Windows에서 저장하기

  1. 시작 메뉴에서 "메모장" 검색 후 실행
  2. 코드 붙여넣기 (Ctrl+V)
  3. 파일 → 다른 이름으로 저장
  4. 파일 형식을 "모든 파일"로 변경 (중요! 그래야 .html 확장자가 적용됨)
  5. 파일명: index.html로 저장

macOS에서 저장하기

  1. TextEdit 실행
  2. 포맷 메뉴 → "일반 텍스트 만들기" 선택 (중요! 기본이 서식 있는 텍스트임)
  3. 코드 붙여넣기
  4. index.html로 저장

VS Code 사용 시 (가장 편한 방법)

VS Code가 설치되어 있다면 새 파일을 열고 index.html로 저장 후 코드를 붙여넣으면 됩니다. 미리보기 확장도 설치할 수 있어 브라우저 없이도 확인이 가능합니다.

3단계: 브라우저 확인 후 수정 반복 (10분)

index.html 파일을 더블클릭하면 기본 브라우저로 열립니다. 화면을 보면서 마음에 안 드는 부분을 목록으로 정리한 뒤, 2~3개씩 묶어서 AI에게 수정 요청합니다:

아래 사항을 수정해주세요:
1) 상단 제목 폰트 크기 더 크게
2) 서비스 섹션 배경을 연한 회색(#f5f5f5)으로 변경
3) 하단에 이메일 연락 버튼 추가
4) 전체 여백 확대 (섹션 간격 좀 더 넓게)

수정 코드를 다시 저장하고 새로고침(F5)하며 반복합니다. 보통 2~3라운드면 원하는 결과가 나옵니다.

4단계: 이미지 준비 (선택사항)

이미지를 사용하려면 사진 파일을 index.html과 같은 폴더에 넣고, AI가 생성한 코드의 파일명과 일치시킵니다:

폴더 구조:
my-site/
  index.html
  profile.jpg    ← 본인 사진
  work1.jpg      ← 작업물 이미지
  work2.jpg

무료 이미지가 필요하다면 Unsplash(unsplash.com)에서 다운로드해서 사용할 수 있습니다. 상업용 사용이 가능하지만, 출처를 표기하는 것이 권장됩니다.

5단계: Netlify 가입 (2분)

netlify.com에서 이메일 또는 GitHub 계정으로 가입합니다. 무료 플랜으로 시작하면 됩니다.

6단계: 폴더 업로드 (2분)

index.html이 들어 있는 폴더를 Netlify 대시보드에 드래그앤드롭합니다.

  • 이미지/아이콘 파일이 있다면 같은 폴더에 함께 있어야 정상 표시됩니다.
  • 파일 하나가 아닌 폴더 전체를 드래그앤드롭해야 합니다.

배포가 완료되면 *.netlify.app 주소가 발급됩니다. 이 주소로 누구나 접속할 수 있습니다.

7단계: URL 공유 및 이름 정리

발급된 주소를 SNS, 명함, 이메일 서명에 사용합니다. Netlify 대시보드에서 Site settings → Change site name으로 URL 앞부분을 더 기억하기 쉬운 이름으로 바꿀 수 있습니다:

기본: random-string-123456.netlify.app
변경: honggildong-portfolio.netlify.app

배포 후 수정 방법

사이트를 업데이트하려면 수정된 파일을 다시 Netlify에 업로드합니다:

  1. Netlify 대시보드에서 해당 사이트 선택
  2. Deploys 탭 클릭
  3. 수정된 폴더를 다시 드래그앤드롭
  4. 자동으로 재배포 완료

자주 막히는 문제와 해결법

  • 파일 더블클릭 시 메모장으로 열림: 파일 우클릭 → 연결 프로그램 → Chrome/Edge로 변경
  • 배포 후 화면 깨짐: 파일만이 아니라 폴더 전체를 업로드했는지 확인
  • 수정 반영 안 됨: 브라우저 강력 새로고침(Ctrl+Shift+R) 또는 Netlify에서 새 배포 실행
  • 이미지가 보이지 않음: 이미지 파일이 index.html과 같은 폴더에 있는지, 파일명 대소문자가 정확한지 확인
  • 모바일에서 화면 이상: AI에게 "모바일 반응형"을 명시해서 재요청

주의사항

  • AI가 만든 문구에 개인정보가 과도하게 들어가지 않도록 확인하세요.
  • 이미지와 아이콘의 라이선스를 배포 전에 확인하세요.
  • 연락처(이메일, 전화번호) 오기입 여부를 반드시 확인하세요.
  • 무료 플랜 정책은 시점에 따라 변경될 수 있습니다.
핵심은 한 번에 완벽한 결과를 만들려는 것이 아니라, 30분 내 공개 가능한 버전을 만든 뒤 점진적으로 개선하는 방식입니다.

마무리

흐름은 단순합니다. AI로 초안 생성 → 로컬 확인 → 수정 반복 → Netlify 배포 → URL 공유. 이 과정만 익히면 개발 경험이 없어도 소개형 홈페이지를 스스로 만들고 운영할 수 있습니다. 처음에는 30분이 걸리지만, 두 번째 사이트는 15분 안에 완성할 수 있게 됩니다.
