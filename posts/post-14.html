코딩 몰라도 AI로 내 포트폴리오 사이트 만들기

게시일: 2025년 11월 15일 · 10분 읽기

취업 준비나 프리랜서 활동을 하다 보면 포트폴리오 사이트가 필요해집니다. 개발 경험이 없어도 AI 도구를 활용하면 기본형 사이트를 빠르게 만들 수 있습니다. 핵심은 코드 작성 자체보다, 원하는 구조를 정확히 설명하고 반복 수정하는 과정입니다.

포트폴리오에 꼭 들어갈 기본 구성

  • 간단한 자기소개
  • 전문 분야 또는 기술 스택
  • 작업물/프로젝트 목록
  • 연락 가능한 채널

이 네 가지가 정리되면 AI에게 지시하기 쉬워지고, 결과물 품질도 안정됩니다.

1단계: 내 정보 먼저 정리하기

프롬프트 전에 텍스트로 정보부터 정리하세요. 아래처럼 간단한 메모만 있어도 충분합니다.

이름: 홍길동
직업: 프리랜서 그래픽 디자이너
전문 분야: 로고 디자인, 브랜딩, SNS 콘텐츠
작업물: A카페 로고 / B쇼핑몰 SNS / C스타트업 인쇄물
연락처: hong@email.com
인스타그램: @honggildong

2단계: AI에게 구체적으로 요청하기

ChatGPT 또는 Claude에 기능과 톤을 함께 지정해 요청합니다. 예: "상단 소개 → 작업물 카드 3개 → 연락처 순서", "모바일 반응형", "한 파일(HTML/CSS/JS)".

막연한 요청보다 레이아웃, 색상, 섹션 수, 모바일 조건을 명시할수록 수정 횟수가 줄어듭니다.

3단계: 결과 확인 후 짧게 반복 수정

생성 코드를 index.html로 저장해 브라우저에서 확인합니다. 이후 한 번에 크게 바꾸기보다 "카드 이미지 영역 확대", "폰트 변경", "스크롤 이동 메뉴 추가"처럼 작은 단위로 수정 요청하는 방식이 빠릅니다.

4단계: 실제 작업물 이미지 반영

AI 초안에는 대체 이미지가 들어가는 경우가 많습니다. 작업물 이미지를 같은 폴더에 넣고 파일명을 맞춰 교체하세요.

5단계: 무료 호스팅으로 배포

Netlify, GitHub Pages, Cloudflare Pages 같은 무료 서비스를 활용하면 공개가 가능합니다. Netlify는 UI가 단순해 입문자가 시작하기 쉽습니다.

  1. 서비스 가입
  2. index.html 포함 폴더 업로드
  3. 발급된 주소 확인
  4. 모바일/데스크톱 점검 후 공유

한계와 체크포인트

  • 복잡한 기능(로그인/결제)은 별도 백엔드가 필요합니다.
  • AI 코드 품질은 요청 문맥에 따라 달라집니다.
  • 저작권, 개인정보 노출, 연락처 오기입을 배포 전에 확인하세요.
무료 플랜의 기능과 한도는 시점별로 변경될 수 있으므로, 배포 전에 각 서비스 공식 안내를 확인하는 것을 권장합니다.

마무리

정리하면 "내 정보 정리 → AI 요청 → 로컬 확인 → 수정 반복 → 배포" 순서입니다. 코드를 완벽히 이해하지 못해도, 명확한 요구와 검수 습관만 있으면 실사용 가능한 포트폴리오 사이트를 만들 수 있습니다.
